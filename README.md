# Консольная утилита для сравнения файлов

## Вы можете сравнить плоские и вложенные файлы .JSON или .yml.

### Установка и запуск в терминале:
```
git clone git@github.com:antonlipilin/DifferenceGenerator.git
make install
make publish
npm link
```
### Справка: 
```
gendiff -h
```
